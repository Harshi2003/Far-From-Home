# Far From Home — Amrita Student Roommate Finder Platform

A mobile-first, cross-platform platform that helps university students discover and connect with compatible roommates based on shared interests, schedules, and housing preferences.

## Features
- **Mobile-first UI (Flutter):** Student-friendly interface to browse roommate profiles and initiate connections
- **Secure Authentication:** Firebase Authentication
- **Profile Management:** Create and update student profiles with preferences and habits
- **Real-time Updates:** Firestore for live data sync
- **Backend Logic:** Cloud Functions for scalable server-side workflows
- **Roommate Matching & Filters:** Custom filtering based on:
  - habits
  - department
  - sleep schedules
  - common interests

## Tech Stack
- **Frontend:** Flutter (cross-platform)
- **Backend:** Firebase (Authentication, Firestore, Cloud Functions)

## Scalability & Future Integration
Designed for scalability within Amrita’s student community with potential integration into university onboarding and housing systems.
